Цей майстер-клас
для тебе, якщо:

  • Ти нещодавно стала мамою і хочеш безпечно відновитися після пологів
  • Тебе турбують болі, діастаз чи підтікання, і ти хочеш позбутися цих проблем
  • Ти хочеш повернутися до активного життя без негативних наслідків для здоров’я
  • Тобі потрібна зрозуміла покрокова програма для відновлення тіла після пологів
  • Ти шукаєш ефективні, але безпечні методи для зміцнення м’язів і покращення самопочуття

Програма

01
Найпоширеніші проблеми жінок після пологів
02
Основні 7-м помилок у відновленні, які зустрчаються у 60% жінок
03
Ключові 5-ть зон, без яких не може бути еффективного відновлення
04
Головні ред флегс та список, чого треба уникати при самостійному відновленні без травм

Авторка
Олена Стоянова

  • Сертифікований інструктор йоги Айєнгара міжнародного рівня (Introductory-II, 2018 р.)
  • Більше 11 років досвіду особистої практики йоги, з них 7 років викладання
  • Спеціаліст з йога-терапії для здоров'я спини, шиї та загального зміцнення тіла
  • Основний досвід роботи - йоги для жінок. Ведення вагітних жінок та безпечне відновлення після пологів.
  • Має досвід двох природних пологів та ефективної практики йоги під час вагітності
  • Успішно справлялась із труднощами, такими як печія, варикоз та втома, зберігаючи активний спосіб життя
  • Мала честь особисто поспілкуватися з Мішелем Оденом, видатним французьким акушером-гінекологом, в Харкові (2017)
